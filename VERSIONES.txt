v1.0.0 — 16/06/2026
═══════════════════════════════════════════════════════════════════
  Lançamento inicial — Display Only

  Funcionalidades:
    ✅ Servidor PC com captura de tela via DXGI (dxcam)
    ✅ Streaming WebSocket binário (JPEG) a 30 FPS
    ✅ APK Android nativo (Kotlin + Jetpack Compose)
    ✅ Cliente web HTML5 (alternativa sem instalar)
    ✅ Barra de conexão com IP configurável
    ✅ Contador de FPS em tempo real
    ✅ Tema dark profissional (#0A0A0B + #6366F1)
    ✅ Qualidade dinâmica auto-ajustável
    ✅ Escala configurável (25% a 100%)
    ✅ Touch input (código pronto, desativado)
    ✅ Teclas de atalho (Esc, Win, Tab, Enter, Space)
    ✅ Ícone do aplicativo (.ico + mipmap Android)
    ✅ Aspect ratio preservado (letterbox)

  Otimizações de performance:
    ✅ JPEG encoding via OpenCV (cv2.imencode) — 5-10x mais rápido
    ✅ Resize via cv2.resize (INTER_AREA) — mais rápido que Pillow
    ✅ Huffman optimization desativada (encode mais rápido)
    ✅ createImageBitmap() no cliente (hardware-accelerated)
    ✅ Qualidade dinâmica baseada em FPS

  Arquivos:
    - MirrorX.exe (Windows, ~72 MB)
    - MirrorX_v1.0.apk (Android, ~5.5 MB, assinado)
    - server.py (fonte Python)
    - client/index.html (cliente web)
    - landing.html (GitHub Pages)
